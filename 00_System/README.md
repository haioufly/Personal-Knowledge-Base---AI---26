## 核心规则

- `02_Knowledge` 强制一个知识一个文件。
- 知识文件必须有唯一 `id` 和标准 frontmatter。
- 标签统一英文 `kebab-case`。
- 重要内容必须通过 `[[双链]]` 连接，不依赖目录树单独定位。
