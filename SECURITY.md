# Security Policy

## Supported Scope

This repository is a documentation-and-template project with small local scripts.

Security-relevant reports may include:

- malicious or dangerous script behavior
- unsafe workflow configuration
- accidental exposure of secrets in examples or templates

## Reporting

Please do not open public issues for sensitive findings.

Contact the maintainer privately with:

- affected file or workflow
- impact
- reproduction steps
- suggested mitigation if available

## Notes

- Do not commit secrets, tokens, personal vault exports, or private data.
- Keep examples synthetic unless explicitly intended for public release.
